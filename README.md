Follow these steps exactly. No coding knowledge needed.

## STEP 1 — Get Your Free Gemini API Key (3 minutes)

1. Open your browser and go to: **https://aistudio.google.com/apikey**
2. Sign in with any **Google / Gmail account**
3. Click the blue **"Create API Key"** button
4. Select **"Create API key in new project"**
5. **Copy the key** — it looks like: `AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXX`
6. Save it somewhere safe (Notepad, Notes app)

✅ This gives you **1,500 free requests per day** — enough for hundreds of users.

---

## STEP 2 — Upload Code to GitHub (5 minutes)

1. Go to **https://github.com** and create a free account (if you don't have one)
2. Click the **"+"** icon → **"New repository"**
3. Name it: `lexindica`
4. Set it to **Public**
5. Click **"Create repository"**
6. On the next page, click **"uploading an existing file"**
7. **Drag and drop ALL files** from the `lexindica-app` folder you downloaded
   - src/App.jsx
   - src/main.jsx
   - index.html
   - package.json
   - vite.config.js
   - vercel.json
   - .gitignore
   - README.md
8. Click **"Commit changes"**

✅ Your code is now on GitHub.

---

## STEP 3 — Deploy on Vercel (5 minutes)

1. Go to **https://vercel.com** and click **"Sign Up"**
2. Choose **"Continue with GitHub"** — sign in with the same GitHub account
3. Click **"Add New Project"**
4. Find your `lexindica` repository and click **"Import"**
5. On the configuration page:
   - Framework Preset: **Vite** (auto-detected)
   - Root Directory: leave as default
6. Expand **"Environment Variables"** section
7. Add this variable:
   - **Name:** `VITE_GEMINI_API_KEY`
   - **Value:** paste your API key from Step 1 (the AIza... key)
8. Click **"Deploy"**
9. Wait 2–3 minutes for the build to complete

✅ Vercel gives you a free URL like: `https://lexindica.vercel.app`

## Troubleshooting

**"API key invalid" error:**
→ Make sure you copied the full key including `AIza` at the start
→ Re-check the Environment Variable in Vercel settings
→ After updating env vars, redeploy: Vercel → Deployments → Redeploy

**App not loading:**
→ Make sure all files were uploaded to GitHub
→ Check Vercel build logs for errors

**"Quota exceeded" error:**
→ Your free 1,500 daily requests are used up
→ Wait until midnight IST for reset, or upgrade Google AI Studio account
